# 実行動画の撮影手順

macOSの `Shift + Command + 5` で画面収録を開始する。通知や個人情報が映らないよう事前に閉じる。

1. Jenkinsジョブ画面と現在の最新ビルド番号
2. ngrokがonlineであり、localhost:8080へ転送している画面
3. GitHub DesktopまたはターミナルでREADMEを1行変更
4. commit・push
5. ngrokに `POST /github-webhook/ 200 OK`
6. Jenkinsの新ビルドが自動開始
7. Console Outputで以下を表示
   - Unit tests
   - Earthquake data output
   - JavaScript dashboard extension
   - Verify outputs
   - Finished: SUCCESS
8. Artifactsの4ファイルを表示
9. `index.html` を開き、地震一覧・検索・M絞り込み・日別グラフを表示

ngrok authtoken、Discord Webhook、GitHubトークン、パスワードは絶対に映さない。
